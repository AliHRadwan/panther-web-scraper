How to use
===============================================================================

1. clone the repo
git clone git@github.com:AliHRadwan/panther-web-scraper.git

2. install the project with its dependencies
composer install

3. install browser drivers (Chrome browser must be installed)
vendor/bin/bdi detect dirvers

4. run the artisan command with a URL form More Carvings as an argument
php artisan scrape:more-cravings "https://www.morecravings.com/en/venues/brasserie-freedom-restaurant-and-bar"

or >>
php artisan scrape:more-cravings "https://www.morecravings.com/en/venues/billiard-bar"

or >>
php artisan scrape:more-cravings "https://www.morecravings.com/en/venues/baroro"

5. you can find the scraped data in a CSV file in storage/app/
panther-web-scraper/storage/app/NAME_TIMESTAMP.csv
